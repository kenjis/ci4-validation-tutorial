# CodeIgniter 4 Validation Tutorial

## Requirement

- CodeIgniter 4.3

## Setup

```console
$ git clone https://github.com/kenjis/ci4-validation-tutorial.git
$ cd ci4-validation-tutorial/
$ composer update
```

```console
$ php spark serve
```
