# CodeIgniter 4 Validation Tutorial

## Requirement

- CodeIgniter 4.3

## Setup

```console
$ git clone git@github.com:kenjis/ci4-validation-tutorial.git
$ cd ci4-validation-tutorial/
$ composer update
```

```console
$ php spark serve
```

## Forms

Navigate to http://localhost:8080/form

Navigate to http://localhost:8080/form2
